# OpenHands Champions List

This repository is the **public, GitHub-native source of truth** for the OpenHands Champions Program.

Champions are **volunteers** recognized for **verifiable artifacts** (projects, write-ups, demos, and/or confirmed participation). If someone claims they are an OpenHands Champion, this repo is where that status is checked.

## Tracks (2026 / Season 1)

- **Hacker**: builders who create a public project using OpenHands.
- **Insider**: trusted community members who participate in product feedback / beta testing.

## How to become a Champion (high level)

1. Complete the requirements for your track(s).
2. Open a PR to this repo **yourself** adding your entry to the relevant table below.
3. Once the PR is reviewed and merged, you’re official for the season.

Details:
- Program overview: [PROGRAM.md](PROGRAM.md)
- How to submit your entry: [CONTRIBUTING.md](CONTRIBUTING.md)
