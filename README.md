# OpenHands Champions

OpenHands Champions is the public contributor directory for everyone who has landed a merged pull request in an OpenHands public repository.

This repository powers a lightweight, searchable contributor directory that:
- tracks merged PR contributors across OpenHands public repos
- preserves a stable GitHub user ID alongside current GitHub profile metadata
- supports self-serve overrides for name, note, and visibility
- highlights recent community momentum through newest contributors and fresh merges
